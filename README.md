# الخطة الاستراتيجية للتعليم في الجفرة 2026-2028

## نظرة عامة

موقع إلكتروني تفاعلي فاخر يعرض الخطة الاستراتيجية الشاملة لتطوير قطاع التعليم في بلدية الجفرة.

### الميزات الرئيسية

✨ **تصميم احترافي وفاخر**
- واجهة عصرية مع تأثيرات حركية سلسة
- ألوان متناسقة (أزرق ملكي، ذهبي، أبيض)
- تصميم مستجيب يعمل على جميع الأجهزة

🌙 **وضع ليلي (Dark Mode)**
- تبديل سهل بين الوضع الفاتح والمظلم
- حفظ تفضيل المستخدم تلقائياً

📱 **تجربة مستخدم متقدمة**
- شريط تقدم القراءة
- قائمة ملاحة ثابتة مع مؤشر القسم النشط
- زر العودة للأعلى
- قائمة جوال (Mobile Menu)

🎯 **محتوى منظم**
- ملخص تنفيذي
- تحليل التحديات الرئيسية
- المحاور الاستراتيجية الأربعة
- التأثير المتوقع
- التوصيات
- الخاتمة

## الملفات والهيكل

```
strategic-plan-website/
├── index.html          # الملف الرئيسي (HTML)
├── style.css           # الأنماط والتصميم (CSS)
├── script.js           # التفاعلات والبرمجة (JavaScript)
├── README.md           # هذا الملف
└── assets/             # مجلد الموارد
    ├── images/         # الصور
    ├── icons/          # الأيقونات
    ├── fonts/          # الخطوط
    └── svg/            # ملفات SVG
```

## الاستخدام

### الفتح المحلي
ببساطة افتح ملف `index.html` في متصفحك:
```bash
# على Windows
start index.html

# على Mac
open index.html

# على Linux
xdg-open index.html
```

### أو استخدم خادم محلي
```bash
# استخدام Python 3
python3 -m http.server 8000

# ثم افتح المتصفح على
http://localhost:8000
```

## النشر المجاني

### GitHub Pages
1. أنشئ مستودع جديد على GitHub
2. ارفع جميع الملفات
3. اذهب إلى Settings → Pages
4. اختر Branch: main
5. سيتم نشر الموقع تلقائياً على: `https://yourusername.github.io/repo-name`

### Netlify
1. اذهب إلى [netlify.com](https://netlify.com)
2. انقر على "New site from Git"
3. اختر مستودعك
4. انقر على Deploy
5. سيتم نشر الموقع تلقائياً

### Cloudflare Pages
1. اذهب إلى [pages.cloudflare.com](https://pages.cloudflare.com)
2. اختر "Create a project"
3. اختر مستودعك
4. انقر على "Save and Deploy"
5. سيتم نشر الموقع تلقائياً

## التخصيص

### تغيير الألوان
عدّل متغيرات CSS في `style.css`:
```css
:root {
    --primary-dark: #1a3a52;
    --primary-blue: #2c5aa0;
    --gold: #d4af37;
    /* ... الخ */
}
```

### تغيير المحتوى
عدّل النصوص مباشرة في `index.html`:
```html
<h1 class="hero-title">عنوانك الجديد</h1>
```

### إضافة صور
ضع الصور في مجلد `assets/images/` وأضفها في HTML:
```html
<img src="assets/images/your-image.jpg" alt="الوصف">
```

## الميزات التقنية

### HTML5
- هيكل دلالي صحيح
- دعم كامل للغة العربية (RTL)
- ميتا تاجز محسّنة للـ SEO

### CSS3
- Grid و Flexbox للتخطيط
- الحركات والتأثيرات (Animations)
- الاستجابة الكاملة (Media Queries)
- متغيرات CSS (CSS Variables)

### JavaScript
- بدون مكتبات خارجية
- Intersection Observer للحركات
- معالج الأحداث الفعّال
- Local Storage للتفضيلات
